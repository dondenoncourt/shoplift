Description: # Create hashtag #

Notes:

curl -X POST --user aaronbartell@gmail.com:poopydiaper -d "parent_id=1" -d "hashtag_value=hat" localhost:3000/hashtags/create.json

POST/hashtags/create.json

Access: FREE
Return: hashtag_value object
Parameters:
  • parent_id:int - id of the item the hashtags is to be added to
  • hashtag_value:string - Hashtag Value
  • json
 # {"id":6,"value":"hat"}
